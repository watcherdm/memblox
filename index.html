Scubed by webspinner

Scubed is a port of the memblox game from gtug campout fame to effect games engine.

Dependencies

Effect Games game engine. http://www.effectgames.com

Install

There are two versions in here, the one we are using is memblox2.js which is an effectgames game. You can put the code into effectgames.com game engine to work on it.
